Programa dividircep:
Assinatura dividircep nome_de_arquivo_de_cep
Exemplo:dividircep cep.dat
cep.dat é nome do arquivo de cep com o formado de linha abaixo.
RUA ARI RODRIGUES                                                       ABRAHÃO ALAB                                                            RIO BRANCO                                                              ACRE                                                                    AC69907190
